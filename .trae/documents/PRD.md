## 1. 产品概述

一个类似 Quizlet 的在线单词卡学习平台，帮助用户通过闪卡学习、测试和游戏化方式记忆知识。
- 目标用户：语言学习者、学生、任何需要记忆知识的用户
- 核心价值：提供高效、有趣、可视化的记忆学习工具

## 2. 核心功能

### 2.1 功能模块

1. **首页/最新动态**: 展示推荐的学习集、最近学习记录、快捷操作入口
2. **学习集浏览**: 浏览和搜索公开的学习集，支持分类筛选
3. **学习集详情**: 查看学习集内的所有单词卡，支持编辑和管理
4. **学习模式**: 支持闪卡模式（翻转卡片）、拼写模式、测试模式、匹配游戏
5. **创建学习集**: 用户可创建自己的单词卡集
6. **个人中心**: 查看用户创建和学习的集合

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 首页 | 顶部导航栏 | 品牌Logo、搜索框、创建按钮、用户头像 |
| 首页 | 英雄区域 | 欢迎语、快捷创建入口、学习统计 |
| 首页 | 学习集推荐 | 展示推荐/最近学习的闪卡集合，卡片网格布局 |
| 首页 | 学习进度 | 显示最近学习进度和待复习卡片数量 |
| 首页 | 学习模式快捷入口 | 快速进入不同的学习模式 |
| 学习集详情 | 集合信息 | 显示标题、描述、卡片数量、作者 |
| 学习集详情 | 单词卡列表 | 以卡片形式展示所有单词/定义对 |

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**: #4255FF (Quizlet 风格蓝色) 为主色，搭配活力橙 #FFCD1F 作为点缀
- **背景**: 浅色模式为主，干净通透的白色背景
- **卡片样式**: 圆角大（16px），柔和阴影，悬浮时有轻微上浮动效
- **字体**: 标题使用 Noto Sans SC（中文）+ Poppins（英文），正文使用系统字体
- **布局**: 顶部导航固定，内容区域最大宽度1200px居中
- **按钮风格**: 圆角按钮，主按钮填充色，次按钮边框样式
- **图标**: 使用简洁的线条图标

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素描述 |
|---------|---------|-----------|
| 首页 | 导航栏 | 左：Logo+文字；中：搜索框；右：创建按钮+头像 |
| 首页 | 英雄区域 | 大标题"今天想学什么？"，副标题，快捷创建卡片 |
| 首页 | 学习集网格 | 3列网格布局，每张卡片包含封面色块、标题、卡片数量、作者 |
| 首页 | 分类标签 | 水平滚动的分类标签：推荐、我的学习集、最近学习 |
| 闪卡模式 | 卡片区 | 居中大卡片，点击翻转，左右箭头切换，进度指示器 |

### 4.3 响应式设计
- 桌面优先设计（1200px宽度）
- 平板：2列网格布局（768px-1024px）
- 手机：单列布局，导航折叠（<768px）
